Regras:

Jogadores 2-4
No começo do jogo todos os jogadoSres têm todos os numeros disponiveis (1-9).
Em cada turno , todos os jogadores têm a sua vez para jogar.
O jogador(a) começa o seu turno por atirar 2 dados. Se a soma dos seus numeros restantes for menos que 6
o jogador atira um unico dado.
Após rodar os dados o jogador usa os numeros que ainda estejam disponiveis, somando-os para igualar ao numero de pintas
que os dados apresentam.

Por exemplo:
Se o jogador rodar os dados e o conjunto de pintas somadas for 8,
pode então usar a combinação de numeros:
  • 8
  • 6,2
  • 7,1
  • 5, 3
  • 5, 2, 1
  • 4, 3, 1

E assim continua o jogo com o objectivo de usar mais numeros possiveis.
Quando nao for possivel usar mais numeros para igualar os resultados dos dados, faz-se a soma dos numeros que ainda nao
foram usados

Por Exemplo:
Imaginando que os numeros restantes eram 2,3,5 a soma seria (2+3+5=10) e esse seria a pontuação do devido jogador.
O jogador com menos pontos ganha, mas, se algum jogador conseguir usar todos os numeros disponiveis, esse jogador
ganha automaticamento a partida.
